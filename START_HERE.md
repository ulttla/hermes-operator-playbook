# Start Here

Hermes is treated as a second operator, not just a spare bot. The useful operating target is mutual support:

- Hermes can review and help recover OpenClaw.
- OpenClaw can review and help recover Hermes.
- gun-wiki gives both operators shared durable context.
- Human approval remains required for risky writes, restarts, updates, credentials, and publishing.

## How to use this playbook

1. Read the [principles](docs/00-principles.md) and [channel model](docs/01-channel-model.md).
2. Use [mutual recovery](docs/02-mutual-recovery.md) when one operator needs review from the other.
3. Use [skills and reusable workflows](docs/04-skills-and-reusable-workflows.md) plus the [skill publication guide](docs/11-skill-publication-guide.md) before publishing any local workflow.
4. Check [redaction and safety](docs/05-redaction-and-safety.md) before moving private operating notes into a public repo.
