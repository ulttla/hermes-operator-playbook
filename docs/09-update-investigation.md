# Update Investigation

Do not treat updates as blind upgrades. Investigate first.

## Read-only flow

1. capture local baseline
2. identify available release and changelog
3. scan issues or community notes
4. compare against local customizations
5. list rollback plan
6. produce go / hold / needs approval verdict

This is especially useful when Hermes reviews an OpenClaw update plan or OpenClaw reviews a Hermes update plan.
