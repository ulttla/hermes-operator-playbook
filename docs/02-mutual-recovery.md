# Mutual Recovery Pattern

The target pattern is bidirectional support:

- OpenClaw helps inspect or repair Hermes when Hermes has a problem.
- Hermes helps inspect or repair OpenClaw when OpenClaw has a problem.
- Both operators use gun-wiki and repo evidence to avoid relying only on chat memory.

## Guardrails

- no automatic version updates
- no unapproved gateway restarts
- no credential printing
- no hidden social or hosting posts
- publish only after validation and human-approved scope
